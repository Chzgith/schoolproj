import pandas as pd 

AllResults = pd.read_csv("AllResults.csv").dropna()

def letter_to_num(data, subject):
    """
    Given a subject and the dataset, changes the letter grades
    into a predetermined numerical scale
    """
    grade_dict = {"N": 0, "B": 1, "2": 2, "3C": 3, "3B": 4, 
                  "3": 5, "3A": 6, "4C": 7, "4B": 8, "4": 9, 
                  "4A": 10, "5C": 11, "5B": 12, "5": 13, 
                  "5A": 14, "6C": 15, "6B": 16, "6": 17, "6A": 18}
    for grade, score in grade_dict.items():
        data.loc[data[subject] == grade, subject] = score
    data[subject] = pd.to_numeric(data[subject])
    return data
   
AllResults = letter_to_num(AllResults, "Re")
AllResults = letter_to_num(AllResults, "Wr")
AllResults = letter_to_num(AllResults, "Ma")

from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error
def plot_line(feature, target, slope, intercept=0):  
    grid = sns.relplot(AllResults, x = feature, y = target)  
    max_x = AllResults[feature].max()  
    min_x = AllResults[feature].min()  

    grid.facet_axis(0, 0).plot([min_x, max_x], [intercept + slope * min_x, intercept + slope * max_x])  
    grid.set(title=f"Slope = {slope:.2f}, Intercept = {intercept:.2f}")  
    return grid  

def linearReg(feature, target):  
    X = AllResults[[feature]]  
    y = AllResults[target]  
    reg = LinearRegression().fit(X, y)  

    print("Model:", " + ".join([f"{reg.intercept_:.2f}"] + [f"{coef:.2f}({X.columns[i]})" for i, coef in enumerate(reg.coef_)]))  
    print("Error:", mean_squared_error(y, reg.predict(X)))  

    plot_line(feature, target, reg.coef_[0], reg.intercept_)

Q. Which variables are important/not important for determining the grades of each subject (Reading, and Math test scores)

Our dataset provides the following variables Regressor variables PP - The pupil premium is additional funding for publicly funded schools in England. EAL - English as an additional language. (0 for False, 1 for True) SEN - Special education needs. (0 for False, 1 for True) Re - Reading grades from age 10 Wr - Writing grades from age 10 Ma - Math grades from age 10 Response variables EngAct - Current grade in English MathsAct - Current grade in Math Given the variables above, we want to test whether the 6 regressors are important/useful in estimating the English/Math grades. For this test, the following assumptions will be made Samples follow a normal distribution (Met by the law of large numbers, n = 910) Independence, which is met given the nature of grades and how they are affected by the individual. Random Sampling Equal variances across the variables With the above requirements in mind, our hypothesis test will be the following (with alpha = 0.05) H0: All coefficients of variables = 0 H1: At least one coefficient is not 0

Analysis: Both models resulted in a MSE around 2.04, indicating that the model may be an accurate representation (does not take into account overfitting) A surprising result is that all 6 regressor variables are significant for reading scores at the 5% level, including math scores. Although there was some evidence that suggested this such as the correlation between Math and English grades, it was still surprising to see that the reading grade model had all variables be significant. On the other hand, the math grade model suggests that the reading and writing grades were not significant at the 5% level for math scores. Reading had a p-value of 0.11, and writing had a p-value of 0.20 which was surprising given the reading model. Unsurprisingly, math grades from age 10 had a p-value of 0 (t* of 15.56). For both models, we reject the null hypothesis generalization without enough variables in mind.

Implications and Limitations

This Analysis can be most helpful to policymakers and educators who are trying to see what can be most helpful for academic performance. Some people who may be excluded from this analysis are people in special education, in this analysis we did not look into special education. Something that could have been a limitation was a lack of variables such as family background, extra curriculors, how much they study, and school support. The second limitation would be the data setting used in this analysis is in a specific time and context, this is pretty much taking an excerpt of data from only one school. All educational environments are different, as this is an English Underperforming School, so that is an inherent flaw. It is good that it has four years of data, however. The last limitation is that it is a linear regression model, the model assumes that it has to be linear for the predictor and outcomes so it is forced to be linear. I would advise people to not use our conclusions as there is too much generalization without enough variables in mind.
